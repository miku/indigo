# Indigo

Inspect files in JSON lines (jsonl, ndjson) format. This tool will count
occurences of keys and will reservoir sample values.

```
$ cat fixtures/1k.jsonl | python indigo.py -s 16 | jq .
```

There are a few top level keys (might change):

* meta
* c (counts)
* s (reservoir sample)
* u (reservoir sample, but reduced to unique values)
* v (the first N unique values)

The counts (c) and reservoir sampled unique values (u) might be the most
useful, currently.

Python 3, stdlib only, no dependencies.

# Usage

```
$ python indigo.py --help
usage: indigo.py [-h] [-s N] [-e NAME] [FILE [FILE ...]]

positional arguments:
  FILE                  files to read, if empty, stdin is used (default: None)

optional arguments:
  -h, --help            show this help message and exit
  -s N, --size N        reservoir sample size (default: 1024)
  -e NAME, --encoding NAME
                        input encoding (for checksum) (default: utf-8)
```
